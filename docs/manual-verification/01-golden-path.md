# 01. Golden Path

## Purpose

Verify the primary end-to-end success path.

## Preconditions

- Environment setup completed
- Test user/input/data prepared
- Current acceptance scenario known

## Procedure

1. Start from `README.md`.
2. Read `00-how-to-use-this-template.md`.
3. Confirm `docs/00-layer-map.md` identifies Source of Truth routing.
4. Confirm `AGENTS.md` explains the agent operating rules.
5. Confirm `docs/08-development-workflow.md` defines Phase execution and report requirements.

## Expected Result

- The flow completes without blocking errors.
- User-visible feedback is clear.
- Final state/output is correct.

## If It Fails

- Identify the failing step.
- Capture logs/errors.
- Link to related Failure Scenario.

## Evidence

- Completed steps
- Screenshots/logs/output
- Remaining risks
